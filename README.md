# Big-O Complexity Visualizer

## Run
```
pip install -r requirements.txt
python app.py
```
Open http://localhost:5000

Drag the slider (or type an exact n) to see O(1), O(log n), O(n), O(n log n), and O(n^2) update live. Toggle series on/off, switch between linear/log Y-axis, and view exact values in the table below the chart.
